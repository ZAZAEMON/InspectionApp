1.Inspection App
- A desktop-based inspection management system built with c# and wpf. 
- Designed to streamline data entry and reporting workflows.
2.key features
- interactive data capture via wpf interface
- modular architecture using services for data processing
- built-in automation scripts for streamlined build and deployment.
3.Tech stack
- language: c#
- ui framework: wpf (windows presentation foundation)
- build tools: batch scripting for automated environment setup
